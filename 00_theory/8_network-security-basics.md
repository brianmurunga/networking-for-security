# Network Security Basics

Securing a network is crucial for protecting data, devices, and availability from unauthorized access or attacks.

---

## 🔐 Security Objectives

- **Confidentiality** – Prevent unauthorized access
- **Integrity** – Prevent data tampering
- **Availability** – Ensure services are accessible

---

## 🧱 Core Security Measures

- **Firewalls** – Control traffic flow (packet filtering or application-level)
- **ACLs** – Enforce traffic policies
- **VPNs** – Encrypt remote access
- **IDS/IPS** – Detect and prevent intrusions
- **Antivirus/Antimalware** – Protect endpoints

---

## 📌 Common Threats

- **DoS/DDoS** – Overwhelm network with traffic
- **Phishing** – Tricking users into revealing info
- **Man-in-the-Middle (MitM)** – Intercepting traffic
- **Malware** – Malicious software

---

## 🧩 Security Best Practices

- Use strong passwords and MFA
- Regular updates/patching
- Network segmentation
- Least privilege access model
- Secure configuration of devices

---

## 📌 Summary
- Network security is multi-layered
- Prevention, detection, and response are key pillars
- Stay updated with threats and tools
